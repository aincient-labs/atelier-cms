# Acknowledgements

Atelier, by AIncient Labs, is built on the shoulders of open source. Thank you to the
projects and the communities behind them.

## Platform

- [Drupal](https://www.drupal.org) — the CMS Atelier is built on
  (GPL-2.0-or-later), created by [Dries Buytaert](https://dri.es/) and built by the Drupal
  community. Drupal is a registered trademark of Dries Buytaert.
- [AI](https://www.drupal.org/project/ai) — the provider-neutral AI framework, together
  with its provider modules (Anthropic, OpenAI, Ollama, …).
- [FlowDrop](https://www.drupal.org/project/flowdrop) — the visual workflow engine that
  powers AIncient's agent flows.
- [Key](https://www.drupal.org/project/key) — secure handling of AI provider credentials.

## Front end

- [assistant-ui](https://www.assistant-ui.com/) — the React chat primitives behind the
  operator console (MIT).
- [React](https://react.dev/) and [Vite](https://vite.dev/) — the console's framework and
  build tooling (MIT).
- [Tailwind CSS](https://tailwindcss.com/) — utility-first styling for the composition
  layer (MIT).

## Typography

- [Fraunces](https://github.com/undercasetype/Fraunces) — the display typeface of the
  Atelier design language (SIL Open Font License 1.1).

And to every contributor to Drupal core and the contrib ecosystem: none of this exists
without you.
